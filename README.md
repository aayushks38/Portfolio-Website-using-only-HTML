# Portfolio Website using only HTML

An assignment-based personal portfolio website built strictly using **HTML5 semantic elements**, without any CSS or JavaScript. Designed to demonstrate a clear understanding of HTML document structure, list nesting, definition lists, tables, forms, hyperlinking, and default browser formatting.

## Features & Sections Implemented
1. **Header**: Name, Student ID, Course, Department, and University details.
2. **About Me**: A 100-150 word professional introduction covering educational background, career goals, personal interests, and hobbies.
3. **Profile Photo**: Embedded profile photo utilizing semantic tags.
4. **Educational Qualifications**: Academic qualifications structured in a clean, readable HTML table.
5. **Technical Skills**: A comprehensive unordered list of technologies and skills.
6. **Programming Languages**: An ordered list ranking proficiency in programming languages.
7. **Academic Subjects**: A definition list (`<dl>`) detailing core academic subjects and their descriptions.
8. **Projects**: Showcase of academic projects with details on descriptions and tech stack.
9. **Achievements**: An unordered list displaying certifications, awards, workshops, and internships.
10. **Hobbies**: Structured nested lists categorization of hobbies.
11. **Useful Links**: Hyperlinks to university website, GitHub, LinkedIn, and email address.
12. **Weekly Timetable**: A highly detailed timetable represented via an HTML table layout matching current semester schedule.
13. **Contact Form**: An interactive form capturing full name, email, phone number, gender, department, DOB, and messages, with functional Submit and Reset buttons.
14. **Footer**: Copyright details and academic year information.

## How to View
Since the website is built strictly in plain HTML5, you do not need any build tool or dependencies to run it. 
Simply open the `index.html` file in any modern web browser, or serve it locally using:
```bash
python -m http.server 8000
```
Then navigate to `https://portfolio-website-using-only-html.vercel.app/` in your browser.
